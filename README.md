## 🎯 Что это?

Phish-scan — это инструмент командной строки для анализа URL-адресов на предмет фишинговой активности. Использует множество факторов для оценки риска: WHOIS данные, SSL сертификаты, анализ контента, проверку редиректов и другие методы.

**Назначение:** Быстрая и точная проверка подозрительных ссылок перед переходом.

---

## ✨ Возможности

### Аналитические модули

| Модуль | Что проверяет | Вес |
|--------|---------------|-----|
| 🌐 **WHOIS анализ** | Возраст домена, регистратора, скрытие данных | 25-35% |
| 🔒 **SSL сертификат** | Наличие, валидность, тип | 20-25% |
| 📝 **Контент анализ** | Формы входа, запрос паролей, чувствительные данные | 15-20% |
| 🔗 **URL анализ** | Подозрительные ключевые слова, спецсимволы | 10-15% |
| 🔄 **Редиректы** | Цепочки перенаправлений, смена домена | 10-15% |

### Режимы работы

- **Интерактивный режим** — пошаговый анализ URL
- **Пакетный режим** — сканирование списка URL из файла
- **Экспорт отчётов** — JSON с детальной информацией

### Визуализация

- 🎨 **Градиентный ASCII баннер**
- 📊 **Анимированный прогресс-бар**
- 🎯 **Цветовая индикация риска**
- 🔄 **Анимация загрузки** (спиннер)
